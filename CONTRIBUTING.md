# Contributing to BlazeKV

Thanks for your interest in contributing! We welcome fixes, features, documentation updates and discussion.

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes and include tests if applicable
4. Commit with clear messages: `git commit -m "feat: add ..."`
5. Push your branch and open a Pull Request

## Code style
- C: Use clear formatting, avoid overly clever macros
- Python: Follow PEP8

## Reporting bugs
Use the issue templates provided in `.github/ISSUE_TEMPLATE`.

## Security
If you discover a security issue, please contact the maintainers directly instead of opening a public ticket.

Thank you for helping improve BlazeKV!
